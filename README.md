# vimconfig

my personal vim configure

# install

```shell
cd ~

git clone git@github.com:sidgwick/vimconfig.git .vim

cd .vim
git submodule update --init

ln -s .vim/home.vimrc .vimrc
```

open vim and run `PluginInstall` command.
